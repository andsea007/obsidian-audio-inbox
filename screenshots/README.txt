# 截图放置说明

将以下截图放到这个文件夹，然后在 Obsidian 重启后会出现在 GitHub README 上：

## 需要 5 张截图

### desktop.png — 桌面端录音界面
1. 电脑上打开 Obsidian
2. 点击左侧 Ribbon 的 🎤 图标，弹出录音窗口
3. 按 `Win + Shift + S` 截取，保存为 `screenshots/desktop.png`

### mobile.png — 手机端悬浮按钮
1. 手机上打开 Obsidian
2. 截取首页（能看到右下角 🎤 悬浮按钮）
3. iPhone：同时按「音量上键 + 电源键」
4. 传到电脑，保存为 `screenshots/mobile.png`

### note.png — 生成的笔记效果
1. 录一段语音（比如"明天下午3点开会，记得买牛奶"）
2. 打开 VoiceNotes 文件夹里生成的笔记
3. 截取笔记内容，保存为 `screenshots/note.png`

### settings.png — 设置面板
1. 打开 Obsidian 设置 → 找到 Audio Inbox
2. 截取设置面板（API Key 记得打码！）
3. 保存为 `screenshots/settings.png`

### summary.png — AI 总结效果
1. 录一段稍长的语音（带待办事项的）
2. 截图 AI 生成的总结和待办事项部分
3. 保存为 `screenshots/summary.png`
